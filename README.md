### Installation Steps



#### 1. Install dependencies

```
pip install opencv-python mediapipe pydirectinput
```

#### 2. install hand_landmarker.task and put it in the folder

#### 3. setup webcam correctly

#### 4. change some settings or code if needed

#### 5. change offset in game to match camera offset (perferably as late as possible to combat the delay)

#### 6. run the main.py

---

### How to Play

#### 1. press space to initialization, put both your hands where you want the slider to be

#### 2. wait for 10 seconds, it should automatically start

#### 3. start playing, press space on the opencv window to stop or start

#### 4. press esc on the opencv window to quit
